Tasks API

Overview

A simple REST API for managing tasks using Express.js. Tasks are stored in tasks.json.

Setup

  • Install dependencies: npm install
  • Run server: npm start or npm run dev
  • Access at: http://localhost:4000

API Endpoints

  • GET /api/tasks — Return all tasks
  • POST /api/tasks — Add a new task (e.g., {"title": "New Task"})
  • PUT /api/tasks/:id — Mark as completed
  • DELETE /api/tasks/:id — Delete a task
